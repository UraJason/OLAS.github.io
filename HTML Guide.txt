#Tags#

<h1></h1> = Heading Tag

<strong></strong> = Bold Tag

<em></em> = Italic Tag

<ol><li></li></ol> = Ordered List Tag (Number)

<ul><li></li></ul> = Unordered List (Bullet point)

<div class="name"></div> = Division Tag (Grouping items/container) CSS

<span></span> = Span Tag (isolate a words in an inline) CSS

<img src="source of image(full path if not int same directory)" alt="(if missing the original image, show text"> Image tag (self closing tag) 

<a href="file path">clickable link text or image, etc</a> = anchor tag, link differennt link
